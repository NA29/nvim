LEADER
------
<Space>        Leader key


CLIPBOARD
---------
<leader>y      Yank selection to system clipboard (normal / visual)
<leader>u      Yank entire file to system clipboard (ggVG"+y)


LSP / CODE NAVIGATION
---------------------
gd             Go to definition (same pane)
gD             Go to definition in vertical split (right)
gi             Go to implementation (same pane)
gI             Go to implementation in vertical split (right)
gr             Find references
gy             Go to type definition
K              Hover documentation


DIAGNOSTICS
-----------
ge             Show diagnostics under cursor
[d             Previous diagnostic
]d             Next diagnostic


SEARCH / FILES
--------------
<C-p>          Telescope: find files
<leader>f      Telescope: live grep

<leader>e      Open mini.files file explorer (focused on current file)

<C-n>          Toggle Neo-tree filesystem (left)
<C-m>          Focus Neo-tree filesystem (left)
<leader>bf     Neo-tree: reveal buffers (floating)


WINDOW / SPLITS
---------------
Splits open to the right and bottom by default

<C-w>v         Vertical split
<C-w>s         Horizontal split
<C-w>q         Close current pane
<C-o>          Jump back in jumplist


EDITING / TEXT MANIPULATION
---------------------------
<A-j>          Move current line down
<A-k>          Move current line up

<A-j> (visual) Move selection down
<A-k> (visual) Move selection up


SCROLLING (SMOOTH)
------------------
<C-d>          Smooth scroll down
<C-u>          Smooth scroll up
<C-f>          Smooth page down
<C-b>          Smooth page up


FILE TREE (NEO-TREE WINDOW)
---------------------------
l / <CR>       Open file / directory
h / <BS>       Navigate up
R              Re-root filesystem
H              Toggle hidden files
I              Toggle gitignored files


MISC
----
<C-Tab>        (Handled by terminal / tmux, not Neovim)
